API Credentials

This page allows you to view your Application Programming Interface (API) Login ID and to create or disable your API Access Tokens

Your API Login ID and Access Tokens are unique pieces of information specifically associated with your CoinMinex account.
However, the API Login ID and Access Tokens are NOT used to for logging into the CoinMinex interface. Access Tokens are only required when setting up an Internet connection between your webservice and CoinMinex gateway. They are used by the CoinMinex gateways.
API Login ID is required to track your client integration code.

IMPORTANT: The API Access Tokens protect your account from unauthorized use and should not be shared with anyone. Be sure to store these values securely on a server separate from your Web server and change the Access Tokens regularly to further strengthen the security of your account.

IMPORTANT: The API Login ID is different than your user login ID. Your user login ID allows you to log into your Member Interface user account. The API Access Tokens grants a access to CoinMinex API. The API Login ID allows us to track your integrations.

You may obtain a new, unique API Login ID and API Access tokens on this page as often as needed.
